Scransition Demo

The javascript adds the class 'scransition' to the elements you've specified. When the element appears on screen, the class is removed and any css associated with the class will be animated away with it. In this way, you can easily animate any CSS parameter which can be animated. This plugin requires CSS transition (or -webkit-transition) support in order for the transition to work. Currently, global support for these features is quite good. Check CanIUse.com to see if your targeted browsers currently have support.
